# Om Parida — Portfolio

A modern, immersive developer portfolio built with Next.js, Three.js, and Framer Motion.

🌐 **Live:** [portfolio.vercel.app](https://portfolio.vercel.app)

---

## Tech Stack

- **Next.js 15** — App Router, TypeScript
- **React Three Fiber** — 3D WebGL scenes
- **Framer Motion** — Animations and transitions
- **Tailwind CSS v4** — Styling
- **Simple Icons** — Brand icons
- **Cormorant Garamond + Outfit** — Typography

## Features

- Interactive 3D hero with WebGL orb, particle cloud and star field
- Floating 3D skills cloud — draggable, filterable by category
- Stacking project cards with video previews
- Animated certifications timeline with slide-in effects
- Looping typewriter code block in the About section
- Custom multi-layer cursor with glow and hover states
- Scroll-aware frosted glass navbar

## Dependencies

```bash
npm install three @react-three/fiber @react-three/drei framer-motion simple-icons lucide-react
npm install -D @types/three
```
